# Norm Lab (一个普通的期中小作业)

大概就是用来规范化学生提交的代码, 以及对文件名和结构进行简单的查重

## 如何开始

**安装 unrar 命令**

```bash
brew install rar # Mac
```

**创建虚拟环境并进入**

```bash
python -m venv myvenv # 创建一个名为 myvenv 的虚拟环境
. myvenv/bin/activate
```

**安装依赖**

```bash
pip install -r requirements.txt
```

**运行**

```
python normlab.py [LabXXX.zip]
```
